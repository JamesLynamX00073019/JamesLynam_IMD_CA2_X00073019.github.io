Android Watch as a Security Key

This project is being completed with assistance from IBM Ireland. The main objective of the project is to allow the user to use their Android Watch as a security token to allow themselves to log on to a desktop or mobile device. This occurs by allowing the transfer of security credentials through Near-Field Communication (NFC).

Once the receiving device has verified that the credentials are correct, it will allow access to the device. A simple use-case would be a user swipeing their watch at their phone in order to log on. This also offers more security over more traditional biometrics such as fingerpring scanning, and facial recognition, which can be falsified.

Work Placement

Claimsure Sláinte Ireland

For the second semester of my third year in college, I entered into an internship with Claimsure Sláinte, which is a subsidiary company of Clanwilliam Health.

The company designed and maintained software to help hospitals and other medical institutions process medical insurance claims. This was done using a programming language known as Health-Level 7, which is the gold standard across the clients.

On the companys side, much of the data processing is done in MySQL, while much of the functionality and security (like that used in their Iguana Translation Engine) is formed in Lua.

I had a great time there and met a lot of good people while learning a lot about the industry too. This is great for me as I intend to enter the medical industry as a software developer.

Unfortunately, I had to stop my internship early due to the COVID-19 pandemic, but I still keep in contact now.
